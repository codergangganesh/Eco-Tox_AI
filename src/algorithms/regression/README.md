# Regression Algorithms

These modules train the regression models used by `train.py`.

- `linear_regression_models.py`: Ridge Regression, Lasso Regression, ElasticNet
- `tree_based_regression_models.py`: Random Forest, Gradient Boosting, XGBoost
- `support_vector_regression.py`: Support Vector Regression with RBF kernel
- `knn_regression.py`: K-Nearest Neighbors Regressor
- `neural_network_regression.py`: MLP neural-network regressor
- `ensemble_regression.py`: Stacking and Voting ensemble regressors

The function names and training workflow are unchanged; only the folder and filenames were clarified.
