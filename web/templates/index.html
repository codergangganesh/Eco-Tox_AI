Problem Statement

Experimental ecotoxicity testing can require controlled aquatic organisms, measured exposure windows, specialized labs, and repeated assays. That makes early screening slow and expensive when many compounds need prioritization.

Project Purpose

The project provides a QSAR-based machine learning workflow that predicts acute aquatic toxicity from six molecular descriptors and helps users compare model reliability before interpreting predictions.

Objectives

Load validated toxicity data, preprocess it consistently, train multiple regression families, select strong candidates, explain important descriptors, and expose the result through a responsive web dashboard.

Core Capability

Users enter CIC0, SM1_Dz_Z, GATS1i, NdsCH, NdssC, and MLOGP values to receive an LC50 estimate, toxicity class, model identifier, and dashboard context for the current training run.

Machine Learning Pipeline

1Data Collection

Loads the local UCI QSAR fish toxicity CSV or downloads it automatically through the UCI repository helper when needed.

2Validation

Applies the expected schema, converts values to numeric format, drops missing values, and removes invalid negative targets.

3Preprocessing

Uses IQR outlier filtering, an 80/20 train-test split with random_state 42, and StandardScaler fitted on training features only.

4Model Training

Trains linear, tree-based, kernel, neighbor, neural-network, stacking, and voting regression models with cross-validation.

5Evaluation

Ranks models using R2, RMSE, MAE, MAPE, and cross-validation statistics, then exports the comparison CSV and plots.

6Prediction

Saves the best estimator and scaler, then serves real-time LC50 predictions through the web app and REST API.

Models Used

Ridge Regression

Linear baseline with L2 regularization to stabilize coefficients when descriptors are correlated.

Lasso Regression

Linear model with L1 regularization that can shrink weaker descriptor coefficients to zero.

ElasticNet

Combines L1 and L2 penalties to balance sparsity and coefficient stability.

Random Forest

Bagged decision-tree ensemble that captures nonlinear descriptor interactions and reports feature importance.

Gradient Boosting

Sequential tree ensemble that fits residual patterns to improve predictive accuracy.

XGBoost

Optimized gradient boosting model with regularization and efficient tree learning.

SVR (RBF)

Kernel-based regressor for nonlinear relationships in small-to-medium descriptor datasets.

KNN Regressor

Similarity-based model that predicts from nearby compounds in descriptor space.

Deep MLP

Multi-layer perceptron regressor trained on scaled descriptors with early stopping behavior.

Stacking and Voting Ensembles

Meta-model and averaging ensembles built from top individual models for stronger aggregate performance.

About EcotoxAI

Understanding the science behind the predictions.

The Problem

Traditional ecotoxicity testing requires culturing aquatic organisms for 48-96 hours per compound. Testing a single chemical can take weeks and cost thousands of dollars.

Our Solution

EcotoxAI uses machine learning trained on 908+ known chemicals to predict the LC50 (lethal concentration) for new compounds in under a second.

The Data

Trained on the UCI QSAR Fish Toxicity dataset — 96-hour LC50 measurements for fathead minnow (Pimephales promelas) with 6 molecular descriptors.

The Models

9 algorithms compared: Ridge, Lasso, ElasticNet, Random Forest, Gradient Boosting, XGBoost, SVR, KNN, and Deep Neural Networks. The best model is selected automatically.

📝 Molecular Descriptors Explained

CIC0 Complementary Information Content — measures molecular complexity and information richness of the chemical graph.
SM1_Dz(Z) Spectral Moment from Barysz matrix weighted by atomic number — captures electronic structure information.
GATS1i Geary Autocorrelation lag 1 weighted by ionization potential — encodes how atomic properties distribute spatially.
NdsCH Count of =CH- fragments — number of doubly-bonded CH groups in the molecule.
NdssC Count of =C< fragments — number of carbon atoms with two double bonds.
MLOGP Moriguchi LogP — lipophilicity measure indicating how easily a compound enters biological membranes. Key driver of aquatic toxicity.
